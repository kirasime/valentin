# Валентинка 14 февраля

Одностраничная валентинка, адаптированная под телефон. Открывается по нажатию на конверт.

## Как изменить текст

Открой `index.html` в редакторе и измени блок с классом `message` — там основной текст поздравления. Подпись — в блоке с классом `sign`.

## Как выложить в интернет

### GitHub Pages (бесплатно)

1. Создай репозиторий на [github.com](https://github.com) → **New repository**. Имя, например: `valentine` или `valentinka`. Не добавляй README — папка уже не пустая.
2. В папке проекта в терминале выполни:

```bash
git init
git add .
git commit -m "Валентинка 14 февраля"
git branch -M main
git remote add origin https://github.com/ТВОЙ_ЛОГИН/ИМЯ_РЕПОЗИТОРИЯ.git
git push -u origin main
```

Замени `ТВОЙ_ЛОГИН` и `ИМЯ_РЕПОЗИТОРИЯ` на свои (например: `https://github.com/ivan/valentine.git`).

3. На GitHub: **Settings → Pages**. В **Source** выбери ветку **main**, папку **/ (root)**. Сохрани.
4. Через 1–2 минуты сайт откроется по адресу:  
   `https://ТВОЙ_ЛОГИН.github.io/ИМЯ_РЕПОЗИТОРИЯ/`

Эту ссылку можно отправить — валентинка откроется на телефоне.

### Другие варианты

- **Vercel**: в папке проекта выполни `npx vercel`, залогинься — получишь ссылку.
- **Netlify**: [app.netlify.com/drop](https://app.netlify.com/drop) — перетащи сюда папку проекта.
